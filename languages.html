What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure and content of web documents using elements and tags. HTML provides the skeleton of web pages, which can then be styled with CSS and made interactive with JavaScript.

Key Points:
  • HTML is not a programming language - it's a markup language
  • It uses tags to structure content
  • It's the foundation of all web pages
  • It works together with CSS (styling) and JavaScript (interactivity)
  • HTML documents are plain text files saved with .html extension

Basic HTML Structure

Every HTML document follows a basic structure that includes essential elements:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Page Title</title> </head> <body> <!-- Content goes here --> </body> </html>

Structure Explanation:

  • <!DOCTYPE html> - Declares the document type and HTML version (HTML5)
  • <html> - Root element containing all content
  • <head> - Contains metadata not displayed on the page (title, styles, scripts, etc.)
  • <body> - Contains all visible content
  • <meta charset="UTF-8"> - Defines character encoding
  • <meta name="viewport"> - Ensures proper rendering on mobile devices
  • <title> - Sets the page title shown in browser tabs

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. It controls the visual appearance, layout, and formatting of web pages, separating content from design.It is used to control the appearance of HTML elements. It defines how elements like headings, paragraphs, and buttons are displayed on a webpage.

Key Points:
  • CSS controls the visual presentation of HTML elements
  • It separates content (HTML) from presentation (CSS)
  • CSS uses a cascading system where styles can inherit and override
  • It enables responsive design and animations

CSS Syntax

CSS consists of rules made up of selectors and declarations. Each declaration contains a property and a value.

Inline CSS is a method of applying styles directly within an HTML element using the style attribute. This approach is used when you want to apply a specific style to a single element only. It is quick to use but not recommended for larger projects because it mixes content and styling, making the HTML code harder to maintain.

Internal CSS refers to writing CSS rules within the <style> tag inside the <head> section of an HTML document. This type of CSS is useful when you need to apply styles to a single web page. It keeps the styling in one place within the file, but it is still limited in reusability across multiple pages.

External CSS is when the CSS code is placed in a separate file with a .css extension and linked to the HTML document using a <link> tag. This method is the most efficient and widely used in professional web development, as it allows consistent styling across multiple pages and keeps the HTML clean.

Three Ways to Add CSS

1. Inline CSS

<p style="color: red; font-size: 16px;">Text</p>

2. Internal CSS

<head> <style> p { color: red; font-size: 16px; } </style> </head>

3. External CSS

<link rel="stylesheet" href="styles.css">

CSS Selectors

A CSS selector is a pattern used to target and select specific HTML elements that you want to style. It tells the browser which elements in the HTML document the CSS rules should be applied to. Selectors can be simple, such as targeting elements by their tag name (like p for paragraphs or h1 for headings), or more specific, such as by class (.classname), ID (#idname), attributes, or even combinations of multiple selectors. By using different types of selectors, developers can apply styles precisely and efficiently to control the appearance of various parts of a web page.

CSS Properties

CSS provides various properties to control the appearance and layout of HTML elements. The color property is used to set the text color, while background-color defines the background color of an element. To control the size of text, the font-size property is used, and font-family specifies which font should be applied. The text-align property helps in aligning the text content, such as left, right, center, or justify. For spacing, margin adds space outside the element's border, and padding adds space inside the element, between the content and the border. The border property allows you to define the width, style, and color of an element’s border. Width and height are used to set the dimensions of an element. The display property controls how an element is displayed on the page, such as block, inline, or flex. Lastly, the position property determines how an element is positioned within the document flow, using values like static, relative, absolute, fixed, or sticky. These properties combined help in designing well-structured and visually appealing web pages.

CSS Box Model

The CSS Box Model is a layout structure that treats every HTML element as a rectangular box made up of four parts: content, padding, border, and margin. The content holds the actual data, padding adds space inside the element, the border wraps around the content and padding, and the margin creates space outside the element. Understanding the box model helps control spacing and layout in web design.

Box Model Components:
  • Content: The actual content area
  • Padding: Space between content and border
  • Border: The border around the element
  • Margin: Space outside the border
.box { width: 200px; height: 100px; padding: 20px; border: 5px solid black; margin: 15px; } /* Box-sizing property */ box-sizing: content-box; /* Default */ box-sizing: border-box; /* Includes padding and border in width/height */

CSS Layout

CSS Layout refers to the way elements are arranged and positioned on a web page using CSS properties. It allows developers to control how content flows and appears across the screen. Common layout techniques include using normal document flow, positioning (static, relative, absolute, fixed, sticky), float, flexbox, and grid. Modern layouts mostly use Flexbox for one-dimensional layouts (like rows or columns) and CSS Grid for two-dimensional layouts (rows and columns together). CSS layout techniques help create responsive, well-structured, and visually balanced web pages that adapt to different screen sizes and devices.

Responsive Design

Responsive design is a web design approach that ensures a website looks and works well on all devices, including desktops, tablets, and mobile phones. It uses flexible layouts, images, and CSS media queries to automatically adjust the content and design based on the screen size or device type. The goal of responsive design is to provide a smooth and user-friendly experience across different screen resolutions without needing separate websites for each device.

Media Queries

3. Operators

An operator is a special symbol or keyword used to perform operations on operands (variables and values). JavaScript includes a wide variety of operators used in different scenarios like arithmetic, comparison, logical evaluation, assignment, etc.

1. Arithmetic operators are used to perform basic mathematical operations such as addition (+), subtraction (-), multiplication (*), division (/), modulus (%), and also include increment (++) and decrement (--) operators. These are commonly used in calculations and loops.

2. Assignment operators are used to assign values to variables. The basic assignment operator is the equals sign (=), but JavaScript also provides compound assignment operators like +=, -=, *=, /=, and %= to perform an operation and assign the result in one step.

3. Comparison operators are used to compare two values and return a Boolean result (true or false). These include == (equal), === (strict equal), != (not equal), !== (strict not equal), > (greater than), < (less than), >= (greater than or equal to), and <= (less than or equal to). 6. Type operators include typeof, which returns the type of a variable as a string (e.g., "string", "number"), and instanceof, which checks whether an object is an instance of a particular constructor or class.

7. Bitwise operators operate on the binary representations of numbers and include & (AND), | (OR), ^ (XOR), ~ (NOT), << (left shift), and >> (right shift). These are less commonly used but are powerful in certain low-level programming tasks.

Move Semantics Explained

Move semantics (C++11) enable efficient transfer of resources:

  • Rvalue References: T&& can bind to temporary objects
  • Move Constructor/Assignment: Transfer resources instead of copying
  • std::move: Casts lvalue to rvalue reference
  • Benefits: Avoid expensive copies of large objects

Key Concepts:

  • Lvalue: Has identity (can take address)
  • Rvalue: Temporary object (no identity)
  • Move operations should be noexcept for optimal performance
  • STL containers and smart pointers support move semantics

Modern C++ Features

Recent C++ standards introduced significant improvements:

  • C++11: auto, lambdas, smart pointers, move semantics
  • C++14: Generic lambdas, return type deduction
  • C++17: Structured bindings, std::optional, filesystem
  • C++20: Concepts, ranges, coroutines, modules

These features make C++ more expressive, safer, and easier to use.
